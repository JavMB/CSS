Explicación del Modelo de Caja

Content (Contenido)

Es el área donde se muestra el contenido real del elemento, como texto, imágenes o cualquier otro tipo de medio. En este ejemplo, es el área naranja con el texto "Contenido de la caja".

Padding (Relleno)

Es el espacio entre el contenido y el borde del elemento. Crea un área transparente alrededor del contenido. En nuestro ejemplo, es el espacio blanco entre el área naranja y el borde azul.

Border (Borde)

Es la línea que rodea el padding y el contenido. Puede tener diferentes estilos, colores y grosores. En este caso, es la línea azul sólida que rodea el área blanca del padding.

Margin (Margen)

Es el espacio exterior al borde. Crea una separación entre el elemento y otros elementos adyacentes. En nuestro ejemplo, es el área blanca que rodea el borde azul, separándolo del fondo gris.
